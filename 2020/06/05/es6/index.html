es6

个人对ES6简单的学习。。。

String拓展(还有很多，只是我过滤了现阶段不需要的)

  1. String.prototype.includes(str) ：返回布尔值，表示是否找到了参数字符串。

  2. String.prototype.startsWith()(str) ：返回布尔值，表示参数字符串是否在原字符串的头部。

  3. String.prototype.endsWith(str) ：返回布尔值，表示参数字符串是否在原字符串的尾部。

  4. String.prototype.repeat(num) ：返回一个新字符串，表示将原字符串重复n次。

    1. num有小数点—->取整
    2. num为负数或者为Infinity(超出JavaScript处理范围的数值)—->报错

  5. String.prototype.padStart(num,str) : 字符串如果不够指定长度，会在前面补全str

    • num:指定长度
    • str:需要补全的字符串

  6. String.prototype.padEnd(num,str) : 在后面补全str

  7. String.prototype.trimStart() :
    消除字符串头部的空格,并且返回一个新的字符串

  8. String.prototype.trimEnd() ：消除字符串尾部的空格,并且返回一个新的字符串

Function拓展

  1. 函数参数默认值
  2. rest参数(形式如: …变量名): 用于获取函数的多余参数
    1.该变量是一个数组
  3. name属性: 返回该函数的函数名
  4. 箭头函数

Proxy(拦截器)

在目标对象之前架设一层“拦截”，外界对该对象的访问，都必须先通过这层拦截，因此可以通过Proxy对外界的访问进行过滤和改写

Proxy 支持的拦截操作

  • target：所拦截的目标对象
  • propKey：所要访问的属性名
  • value：属性值
  1. get(target, propKey, receiver) ：拦截对象属性的读取
  2. set(target, propKey, value, receiver) ：拦截对象属性的设置
  3. has(target, propKey) ：拦截propKey in proxy的操作，返回一个布尔值
  4. deleteProperty(target, propKey) ：拦截delete proxy[proKey]
